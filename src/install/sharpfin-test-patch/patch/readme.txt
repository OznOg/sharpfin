Test Patchfile Readme
=====================

Filename: test.patch
Version: 0.2
Date: 01 October 2007
$Revision: 205 $

The 'test.patch' patchfile is for test purposes only.

It does not install anything on the radio.

The files are unpacked in the radio's /tmp directory.

It prints a diagnostic message on the LCD, and the
radio will then reboot.

REMEMBER that Pressing ENTER will stop the patchserver
