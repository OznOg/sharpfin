Sharpfin Radio Uninstall Script

This script will uninstall ALL of the Sharpfin Base, Webserver, Telnetserver
and any Sharpfin Addons you have installed.

The LCD screen on the radio will give you an indication of its progress:

Uninstall Started	- The disk has been made writeable, and the
			  uninstall process has begun.

Uninstall Removing	- All of the sharpfin packages are being removed.

Completed rebooting...	- Disk is now write protected, and the reboot
			  process is started.

You will have to re-run the patchserver to re-install the Sharpfin programs.
